SmartCalc_v1.0

General information

The calculator has 4 modes of operation: engineering calculator, credit calculator, deposit calculator, graph drawing calculator.
The engineering calculator supports basic math operations and trigonometric functions. Expressions can contain the variable "x" whose value is set after clicking on "equals".
The credit calculator allows you to calculate the monthly payment, overpayment and total payment.
The deposit calculator allows you to calculate the accrued interest, tax and the amount on the deposit at the end of the term. Please note that the calculator uses a tax deduction of 75,000 rubles to calculate the tax.

Installation

To install the calculator, run make install.

Uninstall

To remove the calculator, run make uninstall.
